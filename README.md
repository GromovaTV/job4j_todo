TodoList
=============

This project is for managing current and completed tasks.

### Dependencies

* Java
* Servlets
* JSP(JSTL), HTML, JavaScript(JQuery, Ajax), Bootstrap
* Hibernate + PostgreSQL
* Junit, Hamcrest
* Maven
* Tomcat
* Log4j, Slf4j
* Checkstyle
